ITSUBA

consists of

  • Aluminium body

  • Rubber protectors

  • Gyroscope

  • Battery

  • Acceleration sensor

  • SSD

  • Board

Gyroscope and Acceleration sensors track all shinai movements in 3 dimensions. We store this data and analyze it with the help of machine learning algorithm that defines the type, speed and accuracy of each strike.
